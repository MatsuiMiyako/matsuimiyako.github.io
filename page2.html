Solution

Challenge Overview: The theme is One Piece and involves Base64 encoding and HTML inspection.

Step-by-Step Solution:

  1. Open the challenge and try entering a password.
  2. Click around images—some might be clues.
  3. There is an image with the text "Inspect Me".
  4. Inspect the HTML of that image (right-click, then "Inspect").
  5. You will find an HTML comment revealing the password: "the one piece".
  6. Enter the password to unlock a hidden image.
  7. The new image hints at Base64 encoding.
  8. Some treasure chest images are red herrings (misleading).
  9. Inspect the homepage treasure chest image, revealing a GitHub link in the alt text.
  10. The linked GitHub repository contains a file called "ship log 64".
  11. This hints that the flag is Base64-encoded.
  12. Decode the Base64 string found in the file.
  13. The correct flag is: Th3_0n3_P!ec3_1s_R3al
  14. Enter the flag to complete the challenge!

Hints:

  • Be critical and inspect everything carefully.
  • Click around—some elements react differently.
  • What is visible? What is hidden?
  • Common CTF techniques involve HTML comments and Base64 encoding.
