Types of CTFs

  • Cryptography: Breaking ciphers, encoding/decoding messages.
  • Binary Exploitation: Finding vulnerabilities in executable programs.
  • Reverse Engineering: Analyzing software or code to extract secrets.
  • Web Security: Exploiting website vulnerabilities (e.g., SQL injection, XSS).
  • Forensics: Analyzing digital artifacts (files, logs, network packets).
  • Steganography: Hiding messages inside images, audio, or other files.
  • Pwn: Exploiting system vulnerabilities and privilege escalation.
  • Miscellaneous: Fun puzzles, OSINT, or logic-based challenges.
